# CONTRIBUTING

## Requirements
 - [Nodejs](https://nodejs.dev)
 - [NPM](https://npmjs.com)
 - A webserver (@Aceius uses [web/dev-server](https://github.com/modernweb-dev/web)
 - A GitLab account

## Delveloping
 1. Clone the repo
 2. `cd` into `src/` and run `wds --node-resolve --watch --open` 
 3. Code your commits and then push back into the repo
